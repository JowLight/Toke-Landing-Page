# Toke+ Body Splash — Landing Page

Landing page da **Toke+ Body Splash**, marca de fragrâncias com 4 produtos: Light Dream, Sunset Vanilla, Dark Room e Sensual Fantasy.

## Stack

- HTML5 semântico
- CSS3 (custom properties, animations, media queries)
- JavaScript vanilla (IntersectionObserver, carousel, form handling)
- Google Fonts (Playfair Display + Inter)

## Como rodar

Abra `index.html` no navegador. Não precisa de servidor ou build.

## Estrutura

```
index.html          — Página completa (HTML + CSS + JS inline)
img/                — Imagens dos produtos e banners
images/             — Imagens adicionais (hero, about, etc.)
```

## Responsividade

Breakpoints: 1024px (tablet), 768px (mobile), 480px (small mobile).
